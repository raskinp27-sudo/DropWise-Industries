Abstract

Dropix is a secure, autonomous package drop box designed to prevent porch piracy. Millions of packages are stolen each year due to exposed delivery locations, costing Americans billions of dollars annually and disrupting access to essential items like medication, groceries, and work supplies. Dropix solves this problem by providing a protected container that delivery drivers place packages into, which automatically secures deliveries and prevents theft. The system is designed to be affordable, easy to use, and compatible with homes and apartments, ensuring packages remain safe from the moment they are delivered until the owner retrieves them.

The Solution

Dropix uses a protected drop box that secures itself the moment a package is placed inside — automatically, with no extra steps for the driver.

  • ✓ Protected container — Drivers place packages inside; no porch exposure.
  • ✓ Automatic securing — Locks when a package is detected; no manual action.
  • ✓ Prevents theft — Stops porch piracy instead of only recording it.
  • ✓ Affordable & easy — Designed for homes and apartments, simple to use.
